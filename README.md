# ✈️ Airline Ticket Reservation

A modern web-based Airline Ticket Reservation System that allows users to search flights, select destinations, and book airline tickets through an intuitive and responsive interface.

## 🌐 Live Demo
https://airlineticketreservation.netlify.app/


## 🌍 Overview

The Airline Ticket Reservation System is designed to simplify the flight booking process by providing travelers with an easy-to-use platform for exploring destinations and managing reservations.

## ✨ Features

- Flight search functionality
- Round-trip booking
- One-way booking
- Multi-city booking
- Origin and destination selection
- Passenger and cabin class selection
- Responsive user interface
- User authentication (Login & Signup)
- Modern travel-themed design

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
